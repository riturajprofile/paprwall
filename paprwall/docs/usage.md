# Usage Instructions for PaprWall

## Overview
PaprWall is a modern desktop wallpaper manager that allows users to easily manage and set wallpapers on their desktop while displaying motivational quotes. This guide provides instructions on how to use the application effectively.

## Getting Started
1. **Launching the Application**: 
   - You can start PaprWall by running the following command in your terminal:
     ```
     python -m paprwall
     ```

2. **User Interface**:
   - The main window consists of a header, a preview area for the wallpaper, a quote display, and a sidebar with various controls.

## Features
### Setting a Wallpaper
- To set a wallpaper, you can either:
  - Click the **"Random"** button to fetch a random wallpaper from the internet.
  - Use the **"Browse"** button to select a local image file from your computer.

### Viewing and Refreshing Quotes
- The application displays a motivational quote along with the wallpaper. You can refresh the quote by clicking the refresh button next to the quote display.

### Auto-Rotation
- PaprWall supports auto-rotation of wallpapers:
  - Enable auto-rotation by checking the **"Enable auto-rotation"** checkbox in the sidebar.
  - Set the interval (in minutes) for how often you want the wallpaper to change.

### Custom Image URL
- You can fetch an image from a custom URL:
  - Enter the URL in the **"Custom Image URL"** section and click **"Fetch from URL"**.

### History
- The application maintains a history of wallpapers that have been set. You can view the history in the designated section at the bottom of the main window.

### Settings and Utilities
- Access various settings and utilities from the sidebar, including options to clear history, open the data folder, and uninstall the application.

## Conclusion
PaprWall is designed to enhance your desktop experience with beautiful wallpapers and inspiring quotes. Explore the features and customize your settings to make the most out of the application.
