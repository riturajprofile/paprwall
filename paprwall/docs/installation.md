# Installation Instructions for PaprWall

## Prerequisites

Before installing PaprWall, ensure that you have the following prerequisites:

- Python 3.6 or higher
- pip (Python package installer)

## Installation Steps

### 1. Clone the Repository

You can clone the PaprWall repository from GitHub using the following command:

```bash
git clone https://github.com/yourusername/paprwall.git
cd paprwall
```

### 2. Install Dependencies

Install the required dependencies using pip. You can do this by running:

```bash
pip install -r requirements.txt
```

For development dependencies, you can run:

```bash
pip install -r requirements-dev.txt
```

### 3. Install PaprWall

To install PaprWall, you can use the following command:

```bash
pip install .
```

This command will install the package in your Python environment.

### 4. Verify Installation

After installation, you can verify that PaprWall is installed correctly by running:

```bash
paprwall
```

This should launch the PaprWall application.

## Uninstallation

If you need to uninstall PaprWall, you can do so with the following command:

```bash
pip uninstall paprwall
```
