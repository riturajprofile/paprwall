# PaprWall - Modern Desktop Wallpaper Manager

PaprWall is a feature-rich wallpaper manager that allows users to easily manage and set wallpapers on their desktop. It includes motivational quotes to inspire users while they work.

## Features

- **Modern UI**: A clean and intuitive graphical user interface.
- **Wallpaper Management**: Easily browse and set wallpapers from local files or online sources.
- **Auto-Rotation**: Automatically change wallpapers at specified intervals.
- **Motivational Quotes**: Display inspiring quotes alongside your wallpapers.
- **History Tracking**: Keep track of previously used wallpapers.
