# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-03

### Added
- Initial release of riturajprofile-wallpaper
- Multi-source image fetching (Pixabay, Unsplash, Pexels)
- Local image support
- GTK3 GUI interface
- CLI commands for wallpaper management
- Attribution system with secret key functionality
- Automatic wallpaper rotation
- Configuration management system
- Support for major Linux desktop environments:
  - GNOME/Ubuntu
  - KDE Plasma
  - XFCE
  - MATE
  - Cinnamon
  - LXDE/LXQt
- Desktop overlay for photo credits
- Source selection and weight configuration
- API key management
- Systemd service integration
- Comprehensive documentation

### Features
- Fetch images from multiple API sources
- Use your own local images
- Automatic daily fetching
- Configurable rotation intervals
- Desktop environment auto-detection
- Proper photographer attribution
- Clean, modern GUI
- Command-line interface
- Scheduled background operation

### Documentation
- Complete README with installation guide
- API key setup instructions
- Configuration examples
- Troubleshooting guide
- Development setup guide
